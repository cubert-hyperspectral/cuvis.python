# Test Data

This directory contains test data files for the cuvis.python test suite.

## Required Test Data

- **Aquarium.cu3s** - SessionFile containing 1 measurement for integration tests

### Where to get test data

The Aquarium.cu3s file should be placed in this directory. It will be provided separately by the maintainers.

Note: The .cu3s file may be in .gitignore to keep the repository size manageable.
